## 概述

本文档描述输电线路工频电流的各类故障模式，包括其波形特征、物理含义和识别要点。所有统计数据基于315个真实样本。

### 数据特征

- **采样频率**：3.2kHz
- **数据格式**：CSV（时间戳 + 电流值）
- **采样点数**：约960-1600个（0.3-0.5秒）
- **电流范围**：0A - 5000A（不同电压等级差异大）

### 三段分析方法

算法将整个信号分成三段进行对比分析：
- **前段**（前1/3）：初始状态
- **中段**（中1/3）：变化阶段
- **后段**（后1/3）：最终状态

通过对比三段的RMS值、相对变化率和正弦比等特征来判定故障类型。

## 故障类型与特征

### 1. 跳闸 (Trip)

**物理含义**：线路发生短路、接地故障或过流，保护装置动作，断路器跳闸切断电流。

**波形特征**：
```
前段: 正常工频电流（非近零）
中段: 短路/故障，电流爆发性增长
后段: 衰减至近零（断路器跳闸后）
```

**关键指标**：
- `is_first_near_zero = False`（前段有电流）
- `is_last_near_zero = True`（后段近零）
- `change_mid_first > 2.0`（中段爆发）
- `change_last_mid < 0.5`（后段衰减）

**示例数值**：
```
RMS: 前=50.0A  中=200.0A  后=0.5A
变化: 前中=300%  中后=99.8%  前后=99.0%
```

**典型场景**：
- 相间短路
- 单相接地故障
- 过流保护动作
- 雷击跳闸

**准确率**：73.1%（108个样本）

---

### 2. 合闸成功 (Closing Success)

**物理含义**：断路器合闸操作成功，线路恢复正常供电，后段为稳定工频电流。

**波形特征**：
```
前段: 近零（断路器分闸状态）
中段: 合闸操作，电流从零上升
后段: 稳定工频，完美正弦波
```

**关键指标**：
- `is_first_near_zero = True`（前段近零）
- `is_last_near_zero = False`（后段有电流）
- `sine_ratio_last > 0.998`（后段完美正弦波）
- `change_last_mid >= 0.70`（后/中比值正常）

**示例数值**：
```
RMS: 前=0.0A  中=150.0A  后=200.0A
变化: 前中=∞  中后=33.3%  前后=∞%
正弦比: 后段=0.9995（完美正弦波）
```

**典型场景**：
- 正常合闸操作
- 故障排除后恢复供电
- 备用电源投入

**准确率**：92.0%（98个样本）

---

### 3. 合闸失败 (Closing Failure)

**物理含义**：合闸操作失败，故障未消除，电流显著衰减。

**波形特征**：
```
前段: 近零（断路器分闸状态）
中段: 合闸尝试，电流短暂上升
后段: 显著衰减，未能建立稳定工频
```

**关键指标**：
- `is_first_near_zero = True`（前段近零）
- `is_last_near_zero = False`（后段有电流但衰减）
- `change_last_mid < 0.65`（后段比中段衰减35%以上）
- `change_last_first < 10`（后段上升不足）

**示例数值**：
```
RMS: 前=0.0A  中=200.0A  后=80.0A
变化: 前中=∞  中后=60.0%  前后=∞%
后/中比值: 0.40（显著衰减）
```

**典型场景**：
- 故障未消除，合闸失败
- 断路器机械故障
- 控制回路异常

**准确率**：21.9%（32个样本）

---

### 4. 电流骤升 (Current Surge)

**物理含义**：负荷突然增加或大型设备启动，电流明显上升但前后段均为纯正弦波。

**波形特征**：
```
前段: 正常工频电流（纯正弦波）
中段: 电流过渡上升
后段: 更大工频电流（纯正弦波）
```

**关键指标**：
- `is_first_near_zero = False`（前段非近零）
- `is_last_near_zero = False`（后段非近零）
- `sine_ratio_first >= 0.9`（前段纯正弦波）
- `sine_ratio_last >= 0.9`（后段纯正弦波）
- `change_last_first > 1.5`（后段明显上升）
- `overall_trend = 'rising'`（整体上升趋势）

**示例数值**：
```
RMS: 前=50.0A  中=100.0A  后=150.0A
变化: 前中=100%  中后=50.0%  前后=200.0%
正弦比: 前段=0.995  后段=0.996
```

**典型场景**：
- 大型电动机启动
- 电炉等大负荷设备投入
- 负荷突增

**准确率**：92.9%（28个样本）

---

### 5. 停电/骤降 (Power Off/Sag)

**物理含义**：线路停电操作或负荷突然减少，电流明显下降。

**波形特征**：
```
前段: 正常工频电流
中段: 电流过渡下降
后段: 明显下降或近零
```

**关键指标**：
- `is_first_near_zero = False`（前段非近零）
- `change_last_first < 0.7`（后段明显下降）
- `is_last_near_zero = True/False`（后段可能是近零或小电流）
- `change_mid_first < 1.5`（中段无明显爆发）

**示例数值**：
```
停电场景:
RMS: 前=100.0A  中=50.0A  后=0.0A
变化: 前中=50.0%  中后=100.0%  前后=100.0%

骤降场景:
RMS: 前=100.0A  中=60.0A  后=30.0A
变化: 前中=40.0%  中后=50.0%  前后=70.0%
```

**典型场景**：
- 计划性停电操作
- 负荷突然切除
- 电源侧异常

**准确率**：55.6%（18个样本：10个停电 + 8个骤降）

---

### 6. 上电 (Power On)

**物理含义**：线路初次上电或重启，电流从零上升到正常值。

**波形特征**：
```
前段: 近零（无电状态）
中段: 上电操作，电流从零上升
后段: 稳定工频
```

**关键指标**：
- `is_first_near_zero = True`（前段近零）
- `is_last_near_zero = False`（后段有电流）
- `sine_ratio_last >= 0.99`（后段为正弦波）
- 与合闸成功特征相似

**示例数值**：
```
RMS: 前=0.0A  中=80.0A  后=100.0A
变化: 前中=∞  中后=25.0%  前后=∞%
```

**典型场景**：
- 新设备上电
- 设备重启
- 备用电源投入

**准确率**：归入合闸成功类（30个样本）

---

### 7. 扰动 (Disturbance) ⭐ v14新增

**物理含义**：线路受到轻微干扰但不影响正常供电，前后段电流相近且后段有波动性。

**波形特征**：
```
前段: 正常工频电流（纯正弦波）
中段: 有波动但不影响线路状态
后段: 恢复正常工频电流（与前段相近）且有波动性
```

**关键指标**：
- `is_first_near_zero = False`（前段非近零）
- `is_last_near_zero = False`（后段非近零）
- `0.2 <= change_last_first <= 2.5`（前后段相近，放宽范围）
- `0.5 <= change_mid_first <= 100.0`（中段有波动）
- `std_last / rms_last > 0.3`（后段有波动性，变异系数 > 0.3）

**示例数值**：
```
示例1（轻微扰动）:
RMS: 前=50.0A  中=60.0A  后=52.0A
变化: 前中=20.0%  中后=13.3%  前后=4.0%
后/前比值: 1.04

示例2（较大波动）:
RMS: 前=200.0A  中=260.0A  后=114.0A
变化: 前中=30.0%  中后=56.2%  前后=43.0%
后/前比值: 0.57

示例3（骤降型扰动）:
RMS: 前=484.58A  中=247.43A  后=94.80A
变化: 前中=48.9%  中后=61.7%  前后=80.4%
后/前比值: 0.20
后段变异系数: 0.64（有波动性）
```

**典型场景**：
- 短暂负荷波动
- 开关操作暂态
- 轻微干扰信号
- 骤降型扰动（后段下降但有波动）

**准确率**：100%（基于16个扰动样本测试）

**与跳闸的区别**：
- 跳闸：后段近零（is_last_near_zero=True）
- 扰动：后段非近零且与前期相近（is_last_near_zero=False, 0.2 <= 后/前 <= 2.5）

**与停电/骤降的区别**：
- 停电/骤降：后段明显下降且稳定（后/前 < 0.4，变异系数 < 0.3）
- 扰动：前后段相近且后段有波动（0.2 <= 后/前 <= 2.5，变异系数 > 0.3）

**骤降型扰动判定**：
- 后/前比值可以低至0.2
- 但后段仍有波动（变异系数 > 0.3）
- 区别于停电/骤降的稳定下降特征

### 样本分布

| 故障类型 | 样本数量 | 准确率 |

| 合闸成功 | 98 | 92.0% |
| 跳闸 | 108 | 73.1% |
| 合闸失败 | 32 | 21.9% |
| 电流骤升 | 28 | 92.9% |
| 停电 | 10 | - |
| 电流骤降 | 8 | - |
| 停电/骤降（合并） | 18 | 55.6% |
| 上电 | 30 | 归入合闸成功类 |
| 扰动 | 测试样本 | 100.0% |

### 特征统计

#### 合闸成功
- RMS范围：50A - 500A
- 后段正弦比：> 0.998（完美正弦波）
- 前/后比值：0.0 → 正常值

#### 跳闸
- RMS范围：10A - 2000A
- 前/中比值：1.5 - 5.0（爆发）
- 中/后比值：> 10.0（衰减）

#### 合闸失败
- RMS范围：50A - 500A
- 后/中比值：< 0.65（衰减）

#### 电流骤升
- 前/后比值：1.5 - 5.0（上升）
- 前后段正弦比：> 0.9

#### 停电/骤降
- 前/后比值：> 3.0（下降）
- 后段RMS：近零或明显降低

#### 扰动
- 后/前比值：0.2 - 2.5（放宽范围，支持骤降型扰动）
- 前/中比值：0.5 - 100.0（放宽范围）
- 后段变异系数：> 0.3（有波动性）

## v14算法决策逻辑

### 评分机制

算法为每种故障类型计算一个0-1的评分，选择评分最高的类型作为结果。

### 决策树（优先级）

```
1. 扰动（disturbance）
   ├─ 评分 >= 0.7
   └─ 特征：前后段非近零 + 前后相近（0.5-2.0） + 中段波动（1.0-50.0）

2. 合闸成功（closing_success）
   ├─ 评分 >= 0.7 AND 正弦比 > 0.998
   └─ 特征：前段近零 + 后段完美正弦波

3. 合闸失败（closing_failure）
   ├─ 评分 >= 0.5 AND 后/中 < 0.65
   └─ 特征：前段近零 + 后段显著衰减

4. 跳闸（trip）
   ├─ 最高评分
   └─ 特征：前段非近零 + 中段爆发 + 后段近零

5. 电流骤升（current_surge）
   ├─ 最高评分
   └─ 特征：前段非近零 + 后段上升 + 前后段纯正弦波

6. 停电/骤降（power_off_sag）
   └─ 最高评分
   └─ 特征：前段非近零 + 后段下降
```

### 扰动判定详细逻辑

```python
def calculate_disturbance_score(features):
    score = 0.0

    # 特征1：前后段非近零（区别于合闸成功/失败）
    if not features['is_first_near_zero']:
        score += 0.3
    if not features['is_last_near_zero']:
        score += 0.3

    # 特征2：前后段相近（后/前比值 0.2-2.5）
    # 放宽范围以适应较大幅度的波动，包括骤降型扰动
    if 0.5 <= features['change_last_first'] <= 2.5:
        score += 0.6
    elif 0.2 <= features['change_last_first'] <= 3.0:
        score += 0.4

    # 特征3：中段有波动但不剧烈（0.5 <= 中/前 <= 100.0）
    # 进一步放宽下限，适应骤降型扰动
    if 0.5 <= features['change_mid_first'] <= 100.0:
        score += 0.3

    # 特征4：后段波动性大（区别于停电/骤降）
    # 停电/骤降的后段应该稳定或持续下降，扰动的后段有波动
    if signal is not None:
        n = len(signal)
        std_last = np.std(signal[2*n//3:])
        rms_last = features['rms_last']
        if rms_last > 0 and std_last / rms_last > 0.3:  # 变异系数 > 0.3
            score += 0.3

    return min(score, 1.0)
```

### 算法改进历程

- **v12**：修正骤升/骤降识别，引入正弦比特征
- **v13**：合并停电/骤降，引入后/中比值
- **v14**：新增扰动类型，提高识别准确率

## 使用建议

### 置信度判断

- **置信度 >= 0.9**：结果可信度高
- **0.8 <= 置信度 < 0.9**：结果可信，建议复核
- **置信度 < 0.8**：结果不确定，必须人工复核

### 常见误判场景

1. **合闸失败 vs 合闸成功**
   - 区分度：后/中比值（0.65为阈值）
   - 建议：关注后段衰减程度

2. **跳闸 vs 停电/骤降**
   - 区分度：中段是否爆发
   - 建议：关注中段RMS

3. **扰动 vs 跳闸**
   - 区分度：后段是否近零
   - 建议：关注后段RMS

### 特殊情况处理

1. **数据噪声**：建议滤波或延长采样时间
2. **复合故障**：算法可能识别为主导故障类型
3. **未知故障**：置信度低时标注为未知类型
