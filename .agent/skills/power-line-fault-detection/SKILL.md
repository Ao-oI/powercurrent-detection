# 输电线路工频电流故障识别

## 版本历史

### v22（2025-02-26）
- **优化合闸失败识别**：使用"三段结构+后段衰减"特征
  - 前段近零：front_mean < 15
  - 中段爆发：middle_mean > 50 且 middle/front >= 2.0
  - 后段衰减：back/middle < 0.3（后段明显小于中段）
  - 解决了v21中合闸失败被误判为上电的问题
- **优化跳闸识别**：放宽后段近零阈值，适应连续衰减
  - 后段近零阈值从15A放宽到30A
  - 支持中段爆发或整体大幅下降两种模式
  - 前段非近零（区别于合闸失败）
- **调整判断顺序**：优先判断合闸失败，避免被误判为跳闸
- **准确率**：71.9%（228/317）
  - 跳闸：89.8%（97/108）+13.0%
  - 合闸失败：81.2%（26/32）+34.4%
  - 电流骤升：100%（25/25）
  - 合闸成功：71.4%（40/56）
  - 上电：66.7%（20/30）
  - 扰动：64.5%（20/31）
  - 停电/骤降：需进一步优化

### v21（2025-01-XX）
- 优化扰动识别：增强对中间短暂异常值的鲁棒性
  - 放宽中段稳定性要求，允许短暂异常（change_mid_first < 3.0）
  - 严格前后段幅值相近判断（change_last_first <= 1.5）
  - 允许段数>=3，不限制段数
- 优化骤升识别：更严格地区分骤升和扰动
  - 骤升：后/前 > 1.5，段数=2
  - 扰动：后/前 <= 1.5，允许中段短暂异常
- 修正样本误判：`C0072019060300243_2026-02-26 07_49_09.csv`从"电流骤升"正确识别为"扰动"
- 优化评分函数：调整评分权重，确保扰动和骤升的正确区分
- 优化决策树顺序：优先判断扰动，确保前后段相近的样本不被误判为骤升
- **准确率**：67.5%（214/317）
  - 跳闸：76.9%（83/108）
  - 合闸成功：67.6%（46/68）
  - 骤降：5.6%（1/18）
  - 电流骤升：100%（28/28）
  - 上电：66.7%（20/30）
  - 合闸失败：46.9%（15/32）
  - 扰动：67.7%（21/31）

### v20.16（2025-01-XX）
- 新增上电识别：区分上电和合闸成功
  - 上电：后/前 > 100
  - 合闸成功：2 < 后/前 <= 100
- 优化扰动识别：放宽段数限制，允许段数>=3
  - 增加后段无爆发约束（change_last_mid < 2.0）
  - 提升扰动识别率从9.7%到一定水平
- 优化合闸失败识别：不依赖work_state，只看特征
  - 前段近零 + 后段近零 + 段数>=3
- 优化停电/骤降识别：放宽工作状态和段数限制
- 调整决策树顺序：优先判断跳闸，确保跳闸识别不受影响

### v20.15（2025-01-XX）
- RMS序列平滑：使用滑动窗口中值滤波平滑RMS序列（窗口大小=5个周期）
- 鲁棒的段RMS计算：使用中值代替平均值计算段RMS，避免异常值影响
- 提升对异常频率信号干扰的鲁棒性

## 任务目标
- 本 Skill 用于：识别架空或电缆输电线路的工频电流故障类型，支持1000kV超高压、500kV、220kV、110kV、35kV等多电压等级
- 能力包含：自动判断线路是处于扰动状态，还是发生了跳闸、合闸成功、合闸失败、电流骤升、停电/骤降或上电
- 触发条件：用户提供了工频电流采样数据（采样频率3.2kHz），需要分析线路工作状态

## 注意事项

- **算法原理（v22）**：基于科学分段算法、多特征评分机制和频域特征分析的输电线路工频电流故障识别系统，通过自动识别波形数据段数（1段、2段、3段或多段）来判定故障类型
- **v22核心改进**：
  1. **科学分段算法（v20）**：基于RMS变化率自动识别数据段数，而非强制三等分
  2. **周期对齐的RMS计算**：找到过零点作为周期起始点，避免相位误差
  3. **自适应最小段长度**：`max(1, len(rms_values)/10)`，避免段数过多或过少
  4. **v22决策树优化**：
     - 合闸失败：前段近零 + 中段爆发 + 后段衰减（back/middle < 0.3）
     - 跳闸：前段非近零 + 中段爆发或整体大幅下降 + 后段近零（<30A）
     - 合闸成功：前段近零 + 后段非近零 + 2 < 后/前 <= 100
     - 上电：前段近零 + 后段非近零 + 后/前 > 100
     - 电流骤升：前后段都非近零 + 后/前 > 1.5
     - 扰动：前后段相近（后/前 <= 1.5）+ 非近零
     - 停电/骤降：段数=2 + 后段近零 + 前段非近零
  5. **合闸失败识别优化**：使用后/中比值<0.3作为关键特征，区别于合闸成功/上电
- **准确率说明（v22版本）**：基于317个真实样本测试，总体准确率71.9%
  - 跳闸：89.8%（97/108）
  - 合闸失败：81.2%（26/32）
  - 电流骤升：100.0%（25/25）
  - 合闸成功：71.4%（40/56）
  - 上电：66.7%（20/30）
  - 扰动：64.5%（20/31）
  - 停电/骤降：需进一步优化
- **置信度参考**：当置信度低于0.8时，建议人工复核
- **特征区分**：某些故障类型（如合闸成功/失败、跳闸/停电/骤降）的波形特征相似，区分难度较大

## 故障类型说明（v22）

### 1. 跳闸 (Trip)
- **特征**：前段有电流 → 中段爆发 → 后段衰减至近零（<30A）
- **典型场景**：短路故障、过流保护动作、接地故障等
- **准确率**：89.8%（97/108）

### 2. 合闸成功 (Closing Success)
- **特征**：前段近零 → 后段稳定正弦波 + 中等幅度上升（2 < 后/前 <= 100）
- **典型场景**：线路重合闸成功、开关合闸后恢复正常
- **准确率**：71.4%（40/56）

### 3. 合闸失败 (Closing Failure)
- **特征**：前段近零 → 中段爆发 → 后段衰减（back/middle < 0.3）
- **典型场景**：合闸后立即跳闸、故障点未消除
- **准确率**：81.2%（26/32）
- **v22改进**：使用后/中比值<0.3作为关键特征，解决v21中被误判为上电的问题

### 4. 上电 (Power On)
- **特征**：前段近零 → 后段持续高电流（后/前 > 100）
- **典型场景**：线路首次上电、从停电状态恢复
- **准确率**：66.7%（20/30）

### 5. 电流骤升 (Current Surge)
- **特征**：前后段都非近零 → 后段持续上升（后/前 > 1.5）
- **典型场景**：负载突然增加、并行线路跳闸导致电流转移
- **准确率**：100.0%（25/25）

### 6. 扰动 (Disturbance)
- **特征**：前后段幅值相近（后/前 <= 1.5）+ 非近零 + 允许中间短暂异常
- **典型场景**：短时负荷波动、系统瞬态响应
- **准确率**：64.5%（20/31）

### 7. 停电/骤降 (Power Off/Sag)
- **特征**：段数=2 + 后段近零 + 前段非近零
- **典型场景**：线路停电、负载突然减少
- **准确率**：需进一步优化
